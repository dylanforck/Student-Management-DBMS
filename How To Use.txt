Install all dependencies with:
Bash
pip install -r requirements.txt


(If using the .env approach) create a .env file alongside app.py:
ENV
MYSQL_URL="mysql://root:Password123!@localhost:3306/student_management"
FLASK_SECRET_KEY="some-strong-secret"

Then, in your app.py add at the top:
Python
from dotenv import load_dotenv
load_dotenv()


Run your Flask app as usual:
Bash
python app.py
